# LikeMeiTuan
此项目为仿美团的开源项目，仅供学习交流。

由于时间关系，只完成了整体架构的搭建和首页。
但作为一个完整的安卓项目，除了网络请求的封装，该有的基本都有了。
尤其是架构的分层，可以参考借鉴一下。
